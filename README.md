# -practica-ia
Practica de la asignatura Artificial Intelligence And Open Science In Research Software Engineering

Instalación profesional:
Este proyecto utiliza uv para una gestión de dependencias rápida y reproducible.

Instalar uv: curl -LsSf https://astral.sh/uv/install.sh | sh

Crear y activar entorno: uv venv && source .venv/bin/activate

Instalar: uv pip install -r requirements.txt
